ОБЩИЕ СВЕДЕНИЯ

В каталоге содержатся две программы TVS_VENT и TVS_TOOLS

TVS_TOOLS строит аксонометрические проекции, а так же имеет ряд других полезных функций
Из выбора обрабатываются только обьекты типа: линия,  полилиния, дуга, окружность, незамкнутый сплайн, мультивыноска, текст, мультитекст, остальные игнорируются

TVS_VENT Чертит воздуховоды по всн, считает спецификацию. 
Имеется возможность редактирования начерченых воздуховодов.

Комбо использование данных программ, позволит "Малой кровью" начертить планы по вентиляции и построить 
аксонометрические схемы.


СОВМЕСТИМОСТЬ

TVS_TOOLS-вероятно любой автокад;
TVS_VENT-2010-2012 (x64 x32)



УСТАНОВКА

Ручная

Для программы TVS-VENT


1. добавить папку tvs_vent 2010-2012 x32 или tvs_vent 2010-2012 x64 (x64 или x32 зависит от версии autocad)в 
сервис->настройка..->файл->пути доступа к вспомогательным файлам

2. Загрузить файл StartV.lsp из папки TVS_VEnt (x32 или x64).
 можно просто перетащить в окно Autocad. или ввести команду _appload и выбрать вручную этот файл;
3.  рекомендуется добавить файл StartV.lsp  в автозагрузку 
команда _appload,что бы приложение автоматически подгружалось при запуске, 
а не после нажатия на одну из команд.


Для программы TVS-TOOLS

1. добавить папку tvs_TOOLS в 
сервис->настройка..->файл->пути доступа к вспомогательным файлам
2. Загрузить файл StartL.lsp из папки TVS_TOOLS.
 можно просто перетащить в окно Autocad. или ввести команду _appload и выбрать вручную этот файл;
